# Fake Account Detection System

## Overview
This project detects whether a social media account is fake or genuine using machine learning.

## Features
- TF-IDF text analysis
- Sentiment analysis
- Spam word detection
- Behavioural analysis (followers, following, engagement)
- Streamlit web app

## Tech Stack
- Python
- Scikit-learn
- Pandas
- Streamlit
- TextBlob

## How to Run
1. Install dependencies
2. Run:
   streamlit run app.py
